Cesiumlab通过SHP生成建筑白膜

发表于2022-02-25|更新于2022-02-25|Cesiumlab通过SHP生成建筑白膜
|阅读量:

登录成功后选择左侧*矢量楼快切片*

****输入文件****选择需要转换的shp矢量，SHP需要转换至WGS84（EPSG:4326）坐标系

将面板的属性修改如下

切换至地形高程面板，按如下步骤选择shp所在的DME数据，需要tif格式

切换至贴图效果面板，选择设置

设置floor == 1，选择自定义，分别选择侧面跟顶面的纹理图片，注意贴图的宽度（单位米）

然后选择添加，设置floor == 2 的材质信息

以此类推，一直设置到30多层

对与数量较少的高层楼可以进行适当合并，比如想要设置36-39之间的楼为同一个材质时，可以在设置了floor==35之后，设置条件floor < 40，那么36-39的楼均会被贴上同一种材质

想要40-49为同一个材质时，同样在选择floor<50，那么40-49的楼均会被贴上同一种材质

最后需要设置一个 >= 某个楼层数的条件进行收尾

其中floor < 40，floor < 50，floor >=50只是举例，可以根据不同情况进行拆分，分的越精细模型材质越细化，但是要保证以下几个条件：

a、在满足floor=X条件后，才能设置floor < X

b、只能按顺序设置floor < X，最后设置floor >= Y；不能设置成floor==35, floor >= 36, floor >=40, floor >=50这种组合，这样会把36及以上的矢量均贴为同一材质。

c、屋顶材质的选择最好选择没有边沿的过度的图片，因为贴的材质都只能是正的，而有的楼可能是斜的，这样就会出现瓷砖一样的效果

*文件内提供一些墙体跟房顶的材质可供选择，没有合适的就需要另外找了

*贴图的宽度是指的这张图片对应的模型长度（单位米）

侧面纹理的贴图宽度可以通过图片的宽高比及楼高换算贴图宽度，具体公式为x/y3floor；

（比如图片尺寸为128256，那么当floor=15层时，贴图宽度为 128/2563*15 = 22.5）

顶面纹理的宽度需要依据选择的图片测试找到最佳的效果，同样指这张图片贴的距离（米）；

最后，选择输出文件夹位置，点击确定

等待进度条完成即可

预览数据：选择分发服务>>瓦片服务>>上次保存文件夹的数据

点击图标即可预览，若发现矢量高度不对可以选择14级的地形配套预览

*第二篇*

*修改数据部分*

遇到需要重新生成模型文件时，比如修改了shp，或者某个楼的材质需要单独贴的时候，有以下几种方式：

1、修改了层数，或者新增了矢量时，或者需要更换某个楼层整体的材质时

选择数据处理，处理日志，找到相应处理的版本，选择重新处理

这样贴图信息跟其他属性都在，然后选择新版本的数据，更换新的导出位置，重新制作

对制作完的新数据进行预览查看是否已更新

2、重点房屋需要单独进行贴纹理时

需要两步操作:

第一步，先在完整shp中去除这些数据，重新制作瓦片；

第二步，将选择出的矢量单独导出，然后按第一篇步骤将这条数据进行单独的贴纹理操作，贴图条件不用floor过滤，使用唯一标识id进行过滤，id值为shp里该条数据的id，墙面贴图宽度计算方法跟之前一样x/y3floor，然后将这些重点数据单独导出一份模型瓦片数据。

这样最终成果会是两份
