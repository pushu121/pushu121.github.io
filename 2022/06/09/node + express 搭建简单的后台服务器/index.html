node + express 搭建简单的后台服务器

express框架

Express是基于nodejs的web开发框架。优点是易上手、高性能、扩展性强。

安装

Express是一个基于Node.js的平台，所以在安装Express之前，你得先确保你已经安装了Node.js。

  1. 打开命令行，输入node -v可查看是否有安装node.js。

  1. 创建文件夹并用VSCode打开

  1. 打开新终端 执行 npm init

  1. 起一个包名 其他按回车键一路next

  1. 你会发现多一个package.json文件

  2. 安装expresss 在终端执行 npm i express

  3. 根路径下新建server.js

  4. 新建static/public文件夹 下面新建index.html （后期打包的前端资源放在文件夹下）

  1. 使用 node server启动

  2. 访问localhost:8000
